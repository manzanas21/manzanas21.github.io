Intro a CSS

Selecting by tag name:

Esta etiqueta se utiliza para poner de un determinado color las letras de un texto.
etiqueta { color rgb ( 0-255 , 0-255 , 9-255 ) }
Esta etiqueta se utiliza en el apartado style.
Se ve en todos los h2.

Selecting by id, #:

  • id: Esta etiqueta se utilizan para seleccionar especificamente un elemento basado en el contenido del atributo id.

  • #: Esta etiqueta se utiliza para seleccionar entidades en una capa consiste en seleccionar entidades mediante una consulta de atributos utilizando
    la herramienta seleccionar por atributos, la cual nos permite proporcionar una expresión de consulta SQL que se utiliza para seleccionar entidades
    que coincidan con los criterios de selección.

La generación del 98 es el nombre con el que se ha reunido tradicionalmente a un grupo de escritores, ensayistas y poetas españoles que se vieron profundamente afectados por la crisis moral, política y social desencadenada en España por la derrota militar en la guerra hispano-estadounidense y la consiguiente pérdida de Puerto Rico, Guam, Cuba y Filipinas en 1898. Todos los autores y grandes poetas englobados en esta generación nacen entre 1864 y 1876.
Se inspiraron en la corriente crítica del canovismo denominada regeneracionismo y ofrecieron una visión artística en conjunto en La generación del 98. Clásicos y modernos.

Selecting by class:

Estos electores de clase buscan un elemento basado en el contenido de su atributo class.
Class se define como una lista de elementos separados por espacio, y uno de esos elementos debe coincidir exactamente con el nombre de clase dado en el selector.
Sirve para cambiar el color de fondo de un texto.
